## はじめに

本マニュアルは、シフト管理アプリケーションの操作方法を説明します。本システムは、シフトの自動生成・管理・閲覧を効率的に行うためのツールです。

## 第1章 ログインとホーム画面

### 1.1. ログイン

1. ブラウザで `http://18.182.7.110/` にアクセスします
2. **ユーザーID** と **パスワード** を入力します
3. **「ログイン」** ボタンをクリックします

> **注意**: ログインに失敗した場合は、管理者にユーザーIDとパスワードを確認してください。

### 1.2. ホーム画面

ログイン後のホーム画面には、利用可能な機能へのナビゲーションが表示されます。権限レベルによって表示されるメニューが異なります。

## 第2章 シフト自動生成

### 2.1. シフトを生成する

1. ホーム画面から **「シフト自動生成」** をクリックします
2. **「拠点選択」** ドロップダウンから対象の営業所を選択します
3. **「開始日」** と **「終了日」** を入力します
4. シアン色の **「シフトを生成」** ボタンをクリックします

生成が完了すると、業務（行）× 日付（列）のマトリクス形式でシフト案が表示されます。

### 2.2. マトリクスの見方

| 表示 | 意味 |
|---|---|
| 従業員名バッジ（青） | アサインされた従業員 |
| 空のセル | 未アサイン（誰も割り当てられていない） |
| 「📅 休暇登録済み」行 | その日に休暇を取得している従業員 |
| オレンジチップ | 休暇登録済みの従業員名と休暇種別 |

### 2.3. 未アサイン従業員パネル（右側）

画面右端に表示される **「未アサイン」パネル** には、まだシフトに割り当てられていない従業員が日付ごとに表示されます。

**パネルの操作**:

| 操作 | 方法 |
|---|---|
| 従業員を検索 | 上部の検索ボックスに名前を入力 |
| 日付グループを折りたたむ | 日付ヘッダーの **▲** をクリック |
| 日付グループを展開する | 日付ヘッダーの **▼** をクリック |
| パネルを閉じる | **「◄ 閉じる」** ボタンをクリック（マトリクスが全幅表示になります） |
| パネルを開く | 画面右端の **「►」** ボタンをクリック |

### 2.4. シフトを手動で編集する

**ドラッグ&ドロップで従業員をアサインする**:

1. 未アサインパネルの従業員バッジをドラッグします
2. マトリクス上の空きセルにドロップします
3. 従業員がそのセルにアサインされます

**アサイン済みの従業員を移動する**:

1. マトリクス上のアサイン済み従業員バッジをドラッグします
2. 別のセルにドロップします
3. 従業員が移動します

**従業員をアサインから外す（右クリック削除）**:

1. マトリクス上のアサイン済み従業員バッジを **右クリック** します
2. 表示されたメニューの **「削除」** をクリックします
3. セルが空きになり、従業員が未アサインパネルに戻ります

### 2.5. シフトを保存する

編集が完了したら、シアン色の **「シフトを保存」** ボタンをクリックします。

> **注意**: 変更がある場合のみボタンが有効（シアン色）になります。変更がない場合はグレーで無効化されています。

## 第3章 シフト管理・閲覧

### 3.1. シフトを閲覧する

1. ホーム画面から **「シフト管理」** をクリックします
2. 表示したい **日付** または **期間** を選択します
3. **「従業員軸」** または **「業務軸」** を選択します

### 3.2. 表示モードの切り替え

| モード | 説明 |
|---|---|
| 日次ビュー | 特定の1日のシフトを表示 |
| 期間ビュー | 複数日にわたるシフトを表示 |
| 従業員軸 | 行=従業員で表示 |
| 業務軸 | 行=業務で表示 |

### 3.3. シフトを編集する

**空きセルに従業員をアサインする**:

1. 空きセルをクリックします
2. アサイン可能な従業員の候補が表示されます
3. 候補から従業員を選択します

**シフトを削除する**:

1. 削除したいセルをクリックして選択します（複数選択可）
2. `Delete` キーを押します
3. 確認ダイアログで **「削除」** をクリックします

### 3.4. ルールチェックを実行する

**「ルールチェック」** ボタンをクリックすると、現在のシフトがルールに違反していないか確認できます。違反がある場合は、内容と対象の従業員・日付が表示されます。

### 3.5. Excelにエクスポートする

**「Excel出力」** ボタンをクリックすると、表示中のシフトを Excel ファイルとしてダウンロードできます。

## 第4章 休暇管理

### 4.1. 休暇を登録する

1. ホーム画面から **「休暇管理」** をクリックします
2. **「新規登録」** ボタンをクリックします
3. 以下の項目を入力します:
   - **従業員**: 対象の従業員を選択
   - **休暇日**: 休暇を取得する日付
   - **休暇種別**: 公休 / 私用 / 病欠 / 有給 / 希望休 / その他
   - **理由**: 休暇の理由（任意）
4. **「登録」** ボタンをクリックします

> **重要**: 休暇を登録すると、シフト自動生成時にその従業員はその日の業務に割り当てられません。また、シフトマトリクスの「📅 休暇登録済み」行に表示されます。

### 4.2. 休暇を編集・削除する

一覧から対象の休暇を選択し、**「編集」** または **「削除」** ボタンをクリックします。

## 第5章 従業員管理（管理者向け）

### 5.1. 従業員を追加する

1. ホーム画面から **「従業員管理」** をクリックします
2. **「新規登録」** ボタンをクリックします
3. 以下の項目を入力します:
   - **従業員ID**: 社員番号
   - **氏名**: 従業員の氏名
   - **営業所**: 所属営業所
   - **班**: 所属班（東京営業所のみ）
   - **点呼担当**: 点呼担当可能かどうか
4. **「追加」** ボタンをクリックします

### 5.2. 従業員情報を編集する

一覧から対象の従業員の **「編集」** ボタンをクリックして情報を変更します。

### 5.3. 従業員を削除する

一覧から対象の従業員の **「削除」** ボタンをクリックします。

> **注意**: 削除した従業員に紐づくシフトデータは残ります。

### 5.4. CSVから一括インポートする

1. **「CSVインポート」** ボタンをクリックします
2. インポート用の CSV ファイルを選択します
3. 内容を確認して **「インポート実行」** をクリックします

## 第6章 マスタデータ管理（管理者向け）

### 6.1. 業務マスタを管理する

1. ホーム画面から **「マスタデータ管理」** をクリックします
2. **「業務マスタ」** タブを選択します
3. 業務の追加・編集・削除を行います

**業務マスタの主な設定項目**:

| 項目 | 説明 |
|---|---|
| 業務ID | 業務の識別子 |
| 業務名 | 業務の名称 |
| 営業所 | 担当営業所 |
| 業務グループ | 所属グループ |
| 開始時間 / 終了時間 | 業務の時間帯 |
| ペア業務ID | ペアとなる業務（2名同時アサインの場合） |
| 早朝手当 / 深夜手当 | 各種手当の対象フラグ |

### 6.2. 業務グループを管理する

**「業務グループ」** タブで、業務をまとめるグループを管理します。

## 第7章 ルール管理（管理者向け）

### 7.1. シフトルールを確認する

1. ホーム画面から **「統合ルール管理」** をクリックします
2. ルール一覧が表示されます
3. 種別・営業所・キーワードでフィルタリングできます

### 7.2. ルールを追加・編集する

1. **「新規ルール追加」** ボタンをクリックします
2. ルール名、カテゴリ、種別、適用営業所、優先度、強制レベルを設定します
3. `rule_config` に JSON 形式でルールの詳細設定を入力します
4. **「保存」** をクリックします

### 7.3. ルールを無効化する

ルール一覧の **「有効/無効」** トグルをクリックすることで、ルールを一時的に無効化できます。

## 第8章 よくある質問（FAQ）

**Q: シフト生成後に「未アサイン」の従業員が残っています。どうすればよいですか？**  
A: 右側の未アサインパネルから、ドラッグ&ドロップで空きセルにアサインしてください。

**Q: 従業員を右クリックしてもメニューが表示されません。**  
A: アサイン済みの従業員バッジ（名前が表示されているセル）を右クリックしてください。空きセルでは表示されません。

**Q: シフトを保存したのに「シフトを保存」ボタンがシアン色のままです。**  
A: ボタンがシアン色の場合は未保存の変更があります。もう一度クリックして保存してください。グレーになれば保存完了です。

**Q: 休暇を登録したのにシフトマトリクスの休暇行に表示されません。**  
A: シフトを再生成してください。休暇行は生成時に参照されます。また、休暇登録がない日付は休暇行が表示されません。

**Q: ルールチェックで違反が表示されましたが、どうすればよいですか？**  
A: 違反内容を確認し、対象の従業員・日付のシフトを手動で修正してください。`recommended`（推奨）レベルの違反は警告のみで、保存は可能です。
